# Prompt-Base

Developer-focused web tool for refining prompts.

## Tech Stack

- **Frontend**: React, TypeScript, Vite, Tailwind CSS
- **Backend**: Node.js, Express, TypeScript

## Setup

### Backend

1. `cd backend`
2. `npm install`
3. `cp .env.example .env`
4. `npm run dev`

### Frontend

1. `cd frontend`
2. `npm install`
3. `cp .env.example .env`
4. `npm run dev`
